Return a list of Parts objects for all assembly (Sketchup::Entity) objects and recursive children that can be a part.

An entity can be a part when it is a graphic (made up of only faces and edges) and has at least one pair of mirrored faces that are the desired distance apart.

def parts
  parts = []
  assemblies.each do |assembly|
    assembly.graphics.each do |graphic|
      mirrors = graphic.orientations_at_thickness(depth)
      (parts << Vectorize::Part.new(graphic, self)) if mirrors
    end
  end
  parts
end
